## Mandelbrot Visualizer In C, Written With Raylib and OpenMP for multithreading.

Features: 
* CPU-Based rendering of the mandelbrot set, complete with multithreading. 
* Coloring based on number of iterations to escape.
* Zoom level indicator to see how deep you go.
* Zoom in up to 10^16 times with double precision.

Usage:

install raylib with 
> sudo (dnf / apt / your preferred package manager) install raylib

Then
> ./run.sh 

Once the program is running, click your left mouse button and drag across the screen to select the zoom area. Press R to return to the start. <br>
 **Happy exploring!**
